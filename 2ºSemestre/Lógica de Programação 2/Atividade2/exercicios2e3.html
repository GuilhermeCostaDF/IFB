2)Dos seguintes identificadores, quais são válidos e quais não são e por que?

  1. $algumaVariavel // Válido
  2. _algumaVariavel // Válido
  3. 1Variavel // Inválido
  4. alguma_variavel // Válido
  5. algumävariavel // Válido
  6. function // Inválido obs: Palavra reservada
  7. alguma*variavel // Inválido

Um identificador JavaScript deve começar com uma letra, underline (_), ou cifrão ($); os caracteres subsequentes podem também ser números (0-9).

Devido JavaScript ser case-sensitive, letras incluem caracteres de "A" a "Z" (maiúsculos) e caracteres de "a" a "z" (minúsculos).

Podemos usar a ISO 8859-1 ou caracteres Unicode tal como os identificadores å e ü.Também podemos usar as sequências de escape Unicode como caracteres e identificadores.
